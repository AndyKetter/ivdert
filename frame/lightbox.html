baguetteBox简介

  • 纯 JavaScript 实现，无需依赖其他库
  • 支持触摸屏设备手势操作
  • 支持显示标题（说明）
  • 支持响应式图像
  • 支持CSS3过渡
  • 使用 SVG 按钮，没有多余的文件下载
  • 启用 gzip 压缩后仅 2KB
